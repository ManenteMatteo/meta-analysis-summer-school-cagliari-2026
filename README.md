# Summer School sulle Meta-Analisi

Materiali didattici per systematic review, RStudio e introduzione a `metafor`.

## Come usare il repository

1. Apri `meta-analysis-summer-school.Rproj` in RStudio.
2. Installa i pacchetti indicati in `setup.qmd`.
3. Renderizza il sito con:

```bash
quarto render
```

Il sito renderizzato verrà creato nella cartella `docs/`.

## Pubblicazione su GitHub Pages

Imposta GitHub Pages su:

- Branch: `main`
- Folder: `/docs`
