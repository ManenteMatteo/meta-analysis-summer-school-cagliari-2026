# Summer School sulle Meta-Analisi | Cagliari 2026

Materiali didattici per systematic review, basi di RStudio e introduzione a `metafor`.

## Come usare il repository

0. Scarica la repository in .zip.
1. Apri `meta-analysis-summer-school.Rproj` in RStudio.
2. Installa i pacchetti indicati in `setup.qmd`.
3. Renderizza il sito con:

```bash
quarto render
```

Il sito renderizzato verrà creato nella cartella `docs/`.
