Meta-Analisi: Efficacia della Psicoterapia per la Depressione

Author

Matteo Manente


Durata prevista: 45 minuti


Obiettivo: analizzare un dataset di studi primari rispondendo alla seguente domanda di ricerca

La psicoterapia è efficace nel trattamento della depressione negli adulti?


Il dataset contiene studi che confrontano interventi psicoterapeutici con condizioni di controllo.



Materiali

File per l’esercitazione.

Scarica dataset CSV

Scarica Data Dictionary & Protocol

Note

Il data dictionary descrive le variabili del dataset e contiene alcune informazioni importanti per l’analisi.



Scenario

Avete a disposizione un dataset con effect size estratti da studi primari sulla psicoterapia per la depressione.

Alcune informazioni importanti:

  • ogni riga rappresenta un effect size;

  • alcune pubblicazioni possono contribuire con più effect size;

  • l’effect size è una Standardized Mean Difference calcolata in Hedges’ g;

  • valori positivi indicano che la psicoterapia ha prodotto esiti migliori rispetto al controllo.



Consegna

Effettuare una meta-analisi sul dataset per rispondere alla domanda:

La psicoterapia è efficace per trattare la depressione?

Avete piena libertà sulle scelte analitiche.

Potete decidere, per esempio:

  • quale modello stimare;

  • come gestire studi con più effect size;

  • se escludere o mantenere eventuali outlier;

  • se considerare sottogruppi o moderatori;

  • se controllare publication bias o studi influenti.

Non esiste una sola risposta corretta: l’obiettivo è produrre una scelta analitica ragionevole.



Output richiesto

Al termine dei 45 minuti, dovrete riportare solo questi tre elementi:

  1. Effect size pooled
    Ad esempio: Hedges’ g = …

  2. Intervallo di confidenza al 95%
    Ad esempio: 95% CI [ …, … ]

  3. Una frase interpretativa semplice
    Ad esempio:
    “La psicoterapia sembra essere efficace.”
    “I risultati sono inconclusivi.”
    “L’effetto è positivo, ma l’evidenza è eterogenea.”

Domande guida:

  • Tutti gli effect size possono essere trattati come indipendenti?

  • Il modello fixed-effect è plausibile?

  • Quanto pesa l’eterogeneità?

  • Ci sono studi o effect size influenti?

  • Le conclusioni cambiano se fate scelte analitiche diverse?
