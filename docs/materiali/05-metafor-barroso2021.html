Esercizio Meta-Analisi: Math anxiety e achievement


Obiettivo dell’esercizio

In questo esercizio useremo il dataset barroso2021, incluso nel pacchetto psymetadata.

Il dataset contiene risultati da 332 studi, per un totale di 747 effect size, sulla relazione tra ansia matematica e rendimento matematico.

Gli effect size sono già espressi come correlazioni trasformate in Fisher’s z.

L’obiettivo è mostrare come:

  • preparare un dataset meta-analitico;
  • aggregare più effect size provenienti dallo stesso studio;
  • stimare un modello random-effects con metafor;
  • interpretare output, eterogeneità e prediction interval;
  • visualizzare i risultati;
  • fare semplici analisi su publication bias e sensibilità;
  • introdurre una meta-regressione esplorativa.
ImportantMessaggio centrale

Stimare una meta-analisi in R può richiedere una sola riga di codice.

Interpretarla correttamente richiede però una buona systematic review, decisioni metodologiche esplicite e cautela nell’interpretazione.

1. Setup

Installiamo e carichiamo i pacchetti necessari.

# Installare i pacchetti solo se non sono già presenti
# install.packages("metafor")
# install.packages("psymetadata")

library(metafor)
library(psymetadata)

2. Caricare il dataset

data("barroso2021", package = "psymetadata")

dat <- barroso2021

Esploriamo la struttura del dataset.

str(dat)
head(dat)
names(dat)

Alcune variabili importanti:

  • study_id: identificativo dello studio;
  • es_id: identificativo dell’effect size;
  • author: autore/autori dello studio;
  • pub_year: anno di pubblicazione;
  • ni: numerosità campionaria;
  • yi: effect size osservato, già in Fisher’s z;
  • vi: varianza dell’effect size.

3. Una riga = uno studio o un effect size?

Contiamo quanti effect size e quanti studi unici abbiamo.

nrow(dat)
length(unique(dat$study_id))

Verifichiamo quanti effect size sono presenti per studio.

table_effects <- table(dat$study_id)
summary(table_effects)
NoteNota metodologica

Molti studi contribuiscono con più di un effect size.

Il modello ideale sarebbe quindi un modello multilevel o multivariate, ad esempio con rma.mv().

In questa esercitazione introduttiva useremo una strategia più semplice: aggregare gli effect size entro studio, assumendo una correlazione tra effect size pari a rho = 0.50.

4. Breve passaggio sugli effect size

In questo dataset yi è già in Fisher’s z.

La trasformazione Fisher’s z viene spesso usata perché rende le correlazioni più adatte alla modellizzazione meta-analitica.

# Esempio: da correlazione r a Fisher's z
r_example <- -0.30
z_example <- transf.rtoz(r_example)
z_example

# Esempio: da Fisher's z a correlazione r
transf.ztor(z_example)

Per una correlazione, una formula comune per la varianza della Fisher’s z è:

# Varianza approssimata della Fisher's z
n_example <- 100
vi_example <- 1 / (n_example - 3)
vi_example
TipInterpretazione

Nel modello lavoreremo sulla scala Fisher’s z, ma per interpretare i risultati possiamo trasformare la stima finale di nuovo in correlazione r.

5. Aggregare gli effect size entro studio

Creiamo prima un oggetto escalc, cioè un dataset che metafor riconosce come contenente effect size e varianze.

Poiché yi e vi sono già disponibili, usiamo measure = "GEN".

dat_es <- escalc(
  measure = "GEN",
  yi = yi,
  vi = vi,
  data = dat
)

Ora aggreghiamo gli effect size entro studio.

dat_agg <- aggregate(
  dat_es,
  cluster = study_id,
  rho = 0.50
)

Controlliamo il risultato.

nrow(dat_agg)
head(dat_agg)

Aggiungiamo alcune informazioni utili per grafici e meta-regressione.

study_info <- aggregate(
  cbind(pub_year, grade, ni) ~ study_id,
  data = dat,
  FUN = function(x) mean(x, na.rm = TRUE)
)

author_info <- aggregate(
  author ~ study_id,
  data = dat,
  FUN = function(x) x[1]
)

study_info <- merge(study_info, author_info, by = "study_id")

dat_agg <- merge(
  as.data.frame(dat_agg),
  study_info,
  by = "study_id",
  all.x = TRUE
)

head(dat_agg)
WarningAttenzione

L’aggregazione è una semplificazione didattica.

Riduce la dipendenza tra effect size, ma perde informazioni sulla variabilità entro studio.

In un’analisi reale, questa scelta dovrebbe essere giustificata nel protocollo della systematic review.

6. Prima meta-analisi con rma()

Stimiamo un modello random-effects.

res <- rma(
  yi = yi,
  vi = vi,
  data = dat_agg,
  method = "REML"
)

res

Questa è la struttura minima:

rma(yi, vi, data = dat_agg)
NoteCosa stiamo stimando?

Il modello stima l’effetto medio della relazione tra ansia matematica e rendimento matematico.

Poiché gli effect size sono Fisher’s z, anche la stima del modello è inizialmente sulla scala Fisher’s z.

7. Interpretare l’output

Estraiamo alcuni elementi principali.

res$b       # effetto medio stimato in Fisher's z
res$se      # errore standard
res$ci.lb   # limite inferiore IC
res$ci.ub   # limite superiore IC
res$pval    # p-value
res$tau2    # varianza tra studi
res$I2      # percentuale di variabilità dovuta a eterogeneità
res$QE      # test Q di eterogeneità
res$QEp     # p-value del test Q

Trasformiamo la stima da Fisher’s z a correlazione r.

predict(res, transf = transf.ztor)

Come leggere il risultato

  • La stima media indica la relazione media tra ansia matematica e rendimento.
  • Una correlazione negativa indica che maggiore ansia matematica tende ad associarsi a minore rendimento.
  • L’intervallo di confidenza descrive l’incertezza sulla stima media.
  • tau² e I² indicano quanta eterogeneità rimane tra gli studi.
ImportantDomanda interpretativa

Non basta chiedere: l’effetto è statisticamente significativo?

Bisogna chiedere anche:

  • quanto è grande l’effetto?
  • quanto è incerto?
  • quanto varia tra gli studi?
  • l’effetto medio rappresenta bene la letteratura?

8. Fixed-effect vs random-effects

Confrontiamo un modello fixed-effect e un modello random-effects.

res_fe <- rma(
  yi = yi,
  vi = vi,
  data = dat_agg,
  method = "FE"
)

res_re <- rma(
  yi = yi,
  vi = vi,
  data = dat_agg,
  method = "REML"
)

res_fe
res_re

Interpretazione

Il modello fixed-effect assume che tutti gli studi stimino lo stesso effetto vero.

Il modello random-effects assume che gli effetti veri possano variare tra studi.

Nella maggior parte delle meta-analisi in psicologia, il modello random-effects è spesso più realistico, perché gli studi differiscono per campioni, strumenti, contesti e procedure.

9. Forest plot

Con molti studi, un forest plot completo può diventare poco leggibile.

Per scopi didattici, costruiamo un forest plot su un sottoinsieme di studi.

dat_plot <- dat_agg[order(dat_agg$pub_year), ]
dat_plot <- dat_plot[1:30, ]

res_plot <- rma(
  yi = yi,
  vi = vi,
  data = dat_plot,
  method = "REML"
)

forest(
  res_plot,
  slab = paste(dat_plot$author, dat_plot$pub_year),
  transf = transf.ztor,
  xlab = "Correlazione r",
  header = "Studio"
)

Cosa osservare

  • Studi con intervalli di confidenza più larghi hanno stime meno precise.
  • Studi più grandi tendono ad avere intervalli più stretti.
  • L’effetto pooled è la sintesi complessiva, ma non cancella la variabilità tra studi.

10. Prediction interval

L’intervallo di confidenza riguarda l’effetto medio.

Il prediction interval dà un’idea di dove potrebbe cadere l’effetto vero di un nuovo studio simile.

predict(res, transf = transf.ztor)
TipInterpretazione

Se il prediction interval è molto ampio, significa che gli effetti variano molto tra studi.

In quel caso, l’effetto medio può essere corretto, ma poco rappresentativo di tutti i contesti.

11. Funnel plot e publication bias

Produciamo un funnel plot.

funnel(
  res,
  yaxis = "sei",
  xlab = "Fisher's z"
)

Egger-type regression test

regtest(
  res,
  model = "rma",
  predictor = "sei"
)

Interpretazione cauta

Un funnel plot asimmetrico può suggerire publication bias, ma non lo dimostra automaticamente.

Possibili spiegazioni alternative:

  • eterogeneità reale;
  • studi piccoli di qualità diversa;
  • strumenti di misura diversi;
  • popolazioni diverse;
  • errori di codifica o estrazione.

12. Trim-and-fill

Applichiamo il metodo trim-and-fill.

tf <- trimfill(res)

tf
predict(tf, transf = transf.ztor)

Visualizziamo il funnel plot con gli studi imputati.

funnel(
  tf,
  yaxis = "sei",
  xlab = "Fisher's z"
)
WarningAttenzione

Il trim-and-fill non produce automaticamente una stima “corretta” dell’effetto vero.

È meglio interpretarlo come una sensitivity analysis rispetto a uno specifico scenario di asimmetria del funnel plot.

13. Sensitivity analysis

Leave-one-out

Verifichiamo se il risultato dipende in modo eccessivo da un singolo studio.

loo <- leave1out(res)
loo

Influence diagnostics

inf <- influence(res)
inf
plot(inf)

Domande interpretative

  • C’è uno studio che cambia molto la stima complessiva?
  • Ci sono studi particolarmente influenti?
  • Le conclusioni sono stabili?

14. Meta-regressione esplorativa

Come esempio finale, testiamo se l’anno di pubblicazione è associato alla dimensione dell’effetto.

dat_agg$pub_year_c <- dat_agg$pub_year - mean(dat_agg$pub_year, na.rm = TRUE)

res_mod <- rma(
  yi = yi,
  vi = vi,
  mods = ~ pub_year_c,
  data = dat_agg,
  method = "REML"
)

res_mod

Interpretazione

Il coefficiente di pub_year_c indica quanto cambia l’effect size medio per ogni anno di pubblicazione.

Se il coefficiente è negativo, gli studi più recenti tendono a stimare effetti più negativi.

Se il coefficiente è positivo, gli studi più recenti tendono a stimare effetti meno negativi o più positivi.

WarningCautela

La meta-regressione è osservazionale.

Non dimostra causalità e dovrebbe essere interpretata soprattutto come analisi esplorativa, a meno che il moderatore fosse pre-specificato nel protocollo.

Take-home messages

  • rma() permette di stimare una meta-analisi con pochissimo codice.
  • La parte difficile non è solo il codice, ma la qualità delle decisioni prima e dopo il modello.
  • Gli effect size devono essere sulla scala corretta e accompagnati dalla loro varianza.
  • Se ci sono più effect size per studio, bisogna gestire la dipendenza.
  • Effetto medio, eterogeneità e prediction interval vanno interpretati insieme.
  • Publication bias e trim-and-fill richiedono molta cautela.
  • La meta-regressione è utile, ma non sostituisce una buona teoria e un protocollo chiaro.
